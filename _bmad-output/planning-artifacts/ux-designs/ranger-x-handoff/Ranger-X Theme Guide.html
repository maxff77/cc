Tokens — pega en globals.css

Acento conmutable: el cliente puede preferir cyan o magenta como primario. Añade un atributo y dos reglas:

/* <html data-accent="cyan"> o "magenta" */
[data-accent="cyan"]    { --accent: var(--cyan);    --focus: var(--cyan); }
[data-accent="magenta"] { --accent: var(--magenta); --focus: var(--magenta); }
